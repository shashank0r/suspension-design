# suspension-controller-design
Synapse Drive 2025 PS-2 ; design of semi-active suspension controller for quarter car model
1. load "Single_Profile_Check",  Final CSV and the quarter car simulink model in same workspace.
2. In section 1 of Single_Profile_Check to select the road profile to be analysed.
3. Run the first section of  Single_Profile_Check and then run the simulink model.
4. The calculated data would be displayed on the display and scope block
5. Run the second section of Single_Profile_Check to print the calculated RMS sprung mass displacement, maximum sprung mass displacement, RMS jerk, maximum jerk and the comfort score.
6. Run "Final CSV" to fetch data and generate CSV for all profiles
